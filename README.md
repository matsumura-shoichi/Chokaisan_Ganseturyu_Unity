〇Simulation.zipの使用方法

・Zipファイルをダウンロード・展開後、game2フォルダ内のKisakataganseturyu.exeをダブルクリックすると起動します。

・矢印キーで前進後退旋回します。

・右シフトキーで上昇、左シフトキーで下降します。

・Alt+F4キーで終了します。
